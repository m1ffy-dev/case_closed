# detectivegame

A detective mystery game being developed in Godot.

## About

You play as a rookie investigator solving a series of  mysteries through evidence collection, suspect interviews, and deduction.

## Features

- Investigation system
- Dialogue system
- Evidence collection
- Deduction board
- Multiple endings
